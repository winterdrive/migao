翻譯米糕

忘了切換注音？Ctrl+Alt+R 一秒還原。

在任何 Windows 應用程式中，選取打錯輸入法的亂碼，按快捷鍵，就地替換為正確繁體中文。不需要重打，不需要打開終端機。

migao fix "su3cl3"  # → 你好
下載最新版本 繁體中文說明 ⭐ 在 GitHub 上給星

Windows 日常使用流程

從開始選單啟動 Migao Watch，或開機自動啟動。系統匣出現米糕圖示代表運行中。
正常打字。如果忘記切換輸入法，打出了一串注音亂碼（例如 su3cl3）。
Ctrl+A 選取，再按 Ctrl+Alt+R，亂碼就地替換為「你好」。
3 秒內再按 Ctrl+Alt+R 可循環切換備選答案，直到選到正確的字。

包含什麼

系統匣熱鍵

migao-watch 常駐背景，不需要開終端機，在任何應用程式（LINE、瀏覽器、Word）都有效。

CLI 工具

直接執行修正、管道模式、列出支援的輸入法、顯示多個候選答案。

語言模型

47 萬詞條字典 + Bigram 語言模型，用 Viterbi 演算法選出最符合的繁體中文詞序列。

支援的輸入法

bopomofo-daqian

大千標準注音鍵盤。別名：zhuyin、注音。台灣最常見的注音輸入法配置。

pinyin

全拼 QWERTY 鍵盤。適合簡體中文用戶或習慣拼音輸入的使用者。

即將支援

韓文 두벌식、可設定熱鍵、個人化選字記憶。詳見 ROADMAP。
