# Migao I18N Notes

Migao keeps the English README at the repository root and localized READMEs under `docs/`.

## README language files

- `README.md` — English canonical README
- `docs/README.zh-TW.md` — Traditional Chinese
- `docs/README.zh-CN.md` — Simplified Chinese
- `docs/README.ja.md` — Japanese
- `docs/README.ko.md` — Korean

## Translation policy

- Preserve install commands exactly unless the command itself changes.
- Keep the Windows daily workflow prominent: select garbled text, often with `Ctrl+A`, then press `Ctrl+Alt+R`.
- Translate user-facing explanations, but keep command names, file names, and IME identifiers unchanged.
- If a new feature is documented in `README.md`, update all localized README files in the same PR.

## Link layout

Root README links to localized docs with `./docs/...`.

Localized README files link back to English with `../README.md` and to sibling translations with `README.<locale>.md`.
